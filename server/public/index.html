Kiro Credential Pool

            

                
            

        

        
Each credential = 1 Kiro account (free tier = 50 credits/month + 500 bonus). Stored in kiro_credentials.json (gitignored). Tokens auto-refresh ~1 min before expiry. Cooldown after rate-limit, auto-recovers.


        

            
Add credential

            
Run kiro-cli login --license free --use-device-flow on a machine that ran Kiro IDE; then extract from ~/.local/share/kiro-cli/data.sqlite3 (Linux) or ~/Library/Application Support/kiro-cli/data.sqlite3 (macOS):

            sqlite3 data.sqlite3 "SELECT value FROM auth_kv WHERE key='kirocli:odic:token';"
